## 功能

1. 用户捐赠步数，兑换人民币。
2. 每日 top 排行。
3. 活动信息管理，能够动态修改活动介绍，活动图片等信息。
4. 活动基本信息统计，比如总步数统计，总人数统计，每日捐赠排行。

## 代码目录：

- client: 小程序代码
- design: sketch 设计图
- server: 后端源代码（API）， 用的 Rails4
